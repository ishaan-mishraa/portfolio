# Portfolio

This is my personal portfolio website, built to present who I am, what I work on, and the story behind my projects in a clean, space-inspired experience.

## Overview

More than a simple setup page, this portfolio is designed as a complete website experience. It introduces me through a focused hero section, highlights my skills, shows my timeline and experience, and provides a clear way to get in touch. The overall style is meant to feel modern, polished, and a little futuristic while still staying easy to explore.

## What this website includes

- A strong landing section that introduces the portfolio clearly
- A visually engaging animated space background
- Dedicated sections for skills, experience, and contact information
- Smooth navigation across the page
- A responsive layout that works across screen sizes
- A design that reflects my own style and personal branding

## Tech Stack

- React
- Vite
- CSS
- Tailwind CSS

## Project Structure

- `src/App.jsx` - main application shell
- `src/Home.jsx` - landing and intro section
- `src/Skills.jsx` - skills showcase
- `src/TimelineSection.jsx` - experience and timeline section
- `src/Contact.jsx` - contact form and details
- `src/NavBar.jsx` - navigation bar
- `src/SpaceBackground.jsx` - animated background
- `src/SpaceNavigator.jsx` - navigation helper component

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## About this project

This portfolio is open source and available for everyone to explore, learn from, and adapt. At the same time, it is also a personal project that represents my own work, design choices, and presentation style.
